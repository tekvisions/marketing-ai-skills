# Marketing AI Skills

Curated, auto-updating system prompts (skill files) for AI Agents and LLM chat apps, generated by the [Social Signal Playbook](https://socialsignalplaybook.com).

Use these in Claude, ChatGPT, Perplexity, Gemini Gems, Genspark, ChatLLM, Copilot, OpenClaw agent, Hermes Agent, or your own agents.

## Skill Tiers & Methodology

Our skills are structured into three distinct tiers. Each tier uses a different file size and prompting philosophy to best instruct the LLM:

### 1. Personas (Massive Files: ~100KB+)
**Use Case:** Surrogate experts for your "board of directors."
**Length:** Very large. Designed to completely overwrite the LLM's identity.
**Methodology:** We flood the prompt with behavioral tics, deep psychological worldviews, huge dictionaries of localized slang, and hundreds of aggressive `if/then` constraints to prevent the AI from "breaking character".
**Examples:**
- `ai-cmo-agent`
- `brand-community-builder`

### 2. Topic Guides (Medium Files)
**Use Case:** Deep-dives on specific niche strategies.
**Length:** Standard length. Focuses on data density rather than personality mapping.
**Methodology:** We treat the AI like an analyst, handing it a highly-curated reference manual. The file contains enough dense data, statistics, and domain rules to make the AI an expert on that single topic without needing bloated behavioral guardrails.
**Examples:**
- `zero-click-search`
- `community-led-growth`

### 3. Playbook Frameworks (Ultra-Short Files: < 10KB)
**Use Case:** Rigid, step-by-step diagnostic execution.
**Length:** < 100 lines. Highly tactical.
**Methodology:** Utilizing the "Fabric" pattern (strict `IDENTITY`, strict `INPUT`, strict `OUTPUT`), we force the AI into a tightly enclosed execution loop without any overarching philosophy. Anything extra would dilute the prompt and cause the LLM to break formatting.
**Examples:**
- `status-signaling-audit`
- `attention-arbitrage-calculator`
